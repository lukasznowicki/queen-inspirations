Find your inspiration while writing your newest blog article right on your admin screen!

== Description ==

While writing your blog, sometimes you may need some additional inspirations. Queen songs is the best inspiration for me and I hope it will work for you!

This plugin will display random Queen song title in the upper right of your admin panel. Song title is clickable and will guide you to youtube search results with this particular song.

Absolutely must have for every Queen fan and a nice inspiration for all others. Have fun!

== Installation ==

= Automatic installation =

Automatic installation is the easiest. Simply log in to your WordPress admin panel, go to the Plugins menu and click Add New.

In the search field type Queen Inspirations and click Search Plugins. Then you can install it by simply clicking &#8220;Install Now&#8221; button.

= Manual installation =

1. Upload `queen` directory to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. That's all folks, have fun :)

== Requirements ==

This plugin requires WordPress <b>4.0</b> but please use the newest version (never forget to update your WP installation!) and <b>PHP 5.3</b> installed on your server. It is pretty uncommon to offer previous versions of the PHP, however I must note that.
